# OpenVMM service protocol

The protobuf files in this folder are vendored from the [upstream
OpenVMM vmservice.proto file][upstream-vmservice] at the revision pinned
in `versions.yaml`.

They can be updated via the following command:

```shell
make -C src/runtime-rs/crates/hypervisor update-proto
```

The corresponding Rust bindings will be autogenerated by `cargo build`.

[upstream-vmservice]: https://github.com/microsoft/openvmm/blob/00693a5/openvmm/openvmm_ttrpc_vmservice/src/vmservice.proto
